## Prerequisites

Before installing Repo Agents, ensure you have:

- **Bun 1.3.0 or higher**
- **GitHub CLI** (`gh`) installed and authenticated (used for GitHub operations)
- An **Anthropic API key** for Claude

## Usage

Run Repo Agents directly with bunx (no installation required):

```bash
bunx @repo-agents/cli init
```

### Global Installation (Optional)

If you prefer a global install:

```bash
bun install -g @repo-agents/cli
```

Then run commands directly:

```bash
repo-agents init
```

## Verify

Check the CLI version:

```bash
bunx @repo-agents/cli --version
```

## Authentication

Configure API access:

```bash
repo-agents setup
```

This guides you through adding your Anthropic API key as a repository secret.
