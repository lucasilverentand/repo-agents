# Available Agents

Browse our collection of ready-to-use Claude-powered agents. These agents are fully documented with working examples that you can use in your own projects.

<AgentGallery client:load />

## Getting Started

1. **Choose an agent** from the gallery above
2. **Copy the example** to `.github/claude-agents/`
3. **Customize** the instructions for your needs
4. **Compile and deploy** with `gh claude compile`

```bash
# Initialize gh-claude
gh claude init

# Copy an example agent
cp examples/issue-triage.md .github/claude-agents/

# Compile to workflow
gh claude compile --all
```
